# ⚡ 快速部署指南

想要最快速度部署？選擇以下任一方式，5 分鐘內完成！

## 🚀 最快方式：Vercel（推薦）

### 步驟（3 分鐘完成）：

1. **推送到 GitHub**（如果還沒推送）
   ```bash
   git add .
   git commit -m "Ready to deploy"
   git push
   ```

2. **前往 Vercel**
   - 打開 https://vercel.com
   - 用 GitHub 登入

3. **一鍵部署**
   - 點擊 "Add New..." → "Project"
   - 選擇您的 repository
   - 點擊 "Deploy"（所有設定已自動完成）
   - 等待 1-2 分鐘

4. **完成！** 🎉
   - 獲得一個網址，例如：`https://your-project.vercel.app`
   - 網站已上線！

---

## 🌐 第二快：Netlify

### 步驟（3 分鐘完成）：

1. **推送到 GitHub**（如果還沒推送）
   ```bash
   git add .
   git commit -m "Ready to deploy"
   git push
   ```

2. **前往 Netlify**
   - 打開 https://www.netlify.com
   - 用 GitHub 登入

3. **一鍵部署**
   - 點擊 "Add new site" → "Import an existing project"
   - 選擇您的 repository
   - 點擊 "Deploy site"（設定已自動讀取）
   - 等待 1-2 分鐘

4. **完成！** 🎉
   - 獲得一個網址，例如：`https://your-project.netlify.app`
   - 網站已上線！

---

## 📦 最簡單：拖放部署（Netlify）

不需要 GitHub，直接上傳檔案！

### 步驟（2 分鐘完成）：

1. **建置專案**
   ```bash
   npm install
   npm run build
   ```

2. **拖放部署**
   - 前往 https://app.netlify.com/drop
   - 將 `dist` 資料夾拖放到頁面上
   - 立即完成！

3. **完成！** 🎉
   - 獲得一個網址
   - 網站已上線！

---

## 💡 選擇建議

- **想要自動部署** → 選擇 Vercel 或 Netlify（推送到 GitHub 自動更新）
- **想要最快速度** → 選擇 Netlify 拖放部署
- **想要免費且穩定** → 選擇 Vercel（推薦）

部署完成後，打開您的網站，確認：

- ✅ 股票數據可以載入
